### Ruby Playground

Implementation of:

- 2-opt alogrithm for TSP (Travel Salesman Problem)
- Prim's algorithm for MST (Minimum Spanning Tree)

### Development

```
$ bundle install
```

```
-- Testing --
$ bundle exec rspec

-- Linting --
$ bundle exec rubocop
```
